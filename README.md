# 💧 每日喝水提醒 (Daily Hydrate)

一款简洁实用的Windows桌面喝水提醒工具，帮助您养成健康的饮水习惯。

## ✨ 功能特性

- **📊 进度追踪** - 可视化显示每日喝水进度
- **⚙️ 自定义设置** - 自由设定每日喝水量目标
- **⏰ 智能提醒** - Windows系统通知弹窗提醒
- **📝 记录管理** - 自动记录每日喝水数据
- **🎨 简洁界面** - 现代化的图形用户界面

## 📁 项目结构

```
DailyHydrate/
├── main.py              # 主程序入口
├── requirements.txt     # 依赖列表
├── build.bat            # 打包脚本
├── config.json          # 配置文件（自动生成）
├── README.md            # 说明文档
└── src/                 # 源码目录
    ├── __init__.py      # 包初始化文件
    ├── config_manager.py # 配置管理模块
    ├── reminder.py      # 提醒功能模块
    └── gui.py           # 图形界面模块
```

## 📦 安装依赖

```bash
pip install -r requirements.txt
```

## 🚀 运行程序

### 方式一：直接运行Python脚本
```bash
python main.py
```

### 方式二：打包成exe文件
```bash
# 运行打包脚本
build.bat

# 或手动打包
pyinstaller --onefile --windowed --name "DailyHydrate" main.py
```

打包完成后，可执行文件位于 `dist/DailyHydrate.exe`

## 📖 使用说明

### 基本操作
1. **记录喝水** - 点击快捷按钮或输入自定义水量
2. **设置目标** - 在"计划设置"区域修改每日目标
3. **调整间隔** - 设置提醒的时间间隔（分钟）
4. **开关提醒** - 勾选/取消"启用提醒"

### 快捷按钮
- **一杯 (250ml)** - 记录一杯标准水量
- **半杯 (125ml)** - 记录半杯水量
- **大杯 (500ml)** - 记录大杯水量
- **自定义** - 输入任意水量值

### 配置文件
程序会自动创建 `config.json` 配置文件，保存以下信息：
- 每日目标水量
- 默认杯容量
- 提醒间隔
- 当日喝水记录

## 🔧 系统要求

- **操作系统**: Windows 10/11
- **Python**: 3.7+ (如果直接运行Python脚本)
- **依赖库**: 见 requirements.txt

## 📝 注意事项

1. 程序会在后台运行，定时发送提醒通知
2. 每天的记录会在午夜自动重置
3. 如果 `winotify` 库不可用，程序会使用 tkinter 弹窗作为备选方案

## 🎯 健康建议

根据《中国居民膳食指南》建议：
- 成年人每日饮水量：1500-1700ml
- 建议少量多次饮水
- 不要等口渴了再喝水
